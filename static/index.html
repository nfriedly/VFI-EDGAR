EDGAR API

Submissions

/sub/[adsh]

Example: /sub/0000002178-19-000020

Requesting a single submission will automatically include the associated Numbers and Presentation of Statements.

Tags

  • All versions of a tag: /tag/[tag]. Example: /tag/EntityDomain
  • A specific version of a tag: /tag/[tag]?version=[version]. Example: /tag/EntityDomain?version=dei/2018

Raw data

  • Submissions: /sub?offset=0&limit=10
  • Tags: /tag?offset=0&limit=10
  • Numbers: /num?offset=0&limit=10
  • Presentation of Statements: /pre?offset=0&limit=10

Use the offset and optional limit parameters to paginate results: /sub
